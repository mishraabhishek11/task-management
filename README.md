# 📌 Task Management Application

A simple and intuitive **Task Management System** built with **React.js** that allows users to create projects and manage tasks efficiently.

---

## 🚀 Features

### 📁 Project Management

- Create new projects with:
  - Title
  - Description
  - Due Date
- View all created projects
- Organize tasks based on projects

### ✅ Task Management

- Add tasks under a specific project
- Manage tasks individually
- Track progress for each project
- Clean and user-friendly interface

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **State Management:** React Hooks
- **Styling:** Tailwind CSS
- **Build Tool:** Create React App

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

git clone https://github.com/mishraabhishek11/task-management.git

### 2️⃣ Navigate to the project folder

cd task-management

### 3️⃣ Install dependencies

npm install

### 4️⃣ Start the development server

npm run dev

### 5️⃣ Open in browser

http://localhost:5173/

## 🧑‍💻 Usage

1. Create a new project by filling in project details
2. Select a project to view its tasks
3. Add tasks related to that project
